# TicTacToeRuby

A TicTacToe Engine written in ruby.

## Installation

Add this line to your application's Gemfile:

    gem 'TicTacToeRuby'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install TicTacToeRuby

## Usage

Run 'ruby main.rb' or change the input and output in main.rb
